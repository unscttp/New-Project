# New Project

This workspace now contains a lightweight AI Agent MVP and an archived Snake easter egg.

## Main app

- `frontend/`: Next.js chat UI
- `backend/`: FastAPI agent server and tools

## Snake easter egg

- Runtime copy used by the Next.js app: `frontend/public/snake/`
- Archived original static version: `easter-eggs/snake-static/`

If the chat request fails because of a network problem, the UI will ask whether to open the Snake easter egg.
